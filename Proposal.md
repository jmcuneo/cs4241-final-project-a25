# Project:
Game Review Site 4.0

# Description:
Since I am working alone as opposed to in a group, I am going to be extending one of my previous projects from the class, specifically the Game Review Site I did for A2, 3, and 4 (though I am going to build off of my A3 version rather than 4 as I personally found that format more accessible than the Vite version).

There are four main features/updates I intend to add:
  1. Expanding User Profiles:
Currently the only information tied to a user is their reviews and their login info, so I am going to add more data such as a bio, their pronouns, and favourite games, as well as pages to view and edit that data.

  2. Making Other User Reviews and Profiles Viewable:
Currently a user can only view their own reviews but I am going to add a page that lists all users, and make their profile and review viewable to any other user.

  3. Liking Reviews:
With the addition of profiles, I want to add the ability for users to like other user's reviews, which will be displayed on the review. Maybe even a page to see reviews you've liked.

  4. Interactive Game Diagram:
I want to use D3 to create an interactive diagram of both the games each individual user has logged, being able to sort by release date, ratings, et cetera, also being able to select from a few different data viewing formats (bar chart, bubble chart, pie chart, et cetera).

Lastly, beyond this, whilst my current UI is perfectly functional and accessible, I want to do what I can to clean it up a bit.
