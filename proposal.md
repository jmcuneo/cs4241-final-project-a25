## Project Description
Our project will be a nutrition tracker that allows users to log meals and monitor 
their daily calories and macronutrients. The app is inspired by apps like myFitnessPal 
and is meant for those who want to better understand what they are actually eating on a
day to day basis.

The static part of the application will have resource pages on healthy eating and 
example recipes, complete with images and instructions. These pages will be easy to 
navigate and accessible, serving as reference material alongside the tracking system.

The dynamic part of the application will include an interactive meal log and 
nutrition dashboard. Users can choose foods from a preloaded database of common foods 
and recipes. As users add or remove items from their daily log, the app will update 
total calories, protein, carbohydrates, and fat automatically.  This helps show 
the results in real time through charts or progress bars.

On the server side, the project will have user authentication and persistent storage. 
Registered users can sign up, log in, and save their meal history. The backend will 
use Node.js and Express and will keep user accounts, food items, and logged meals in 
a database, likely PostgreSQL. 

## Key Features
- Static recipe/info pages with images and instructions
- Authentication (sign up, login, logout)
- Preloaded food database with calories/macros
- Daily meal log for users
- Dynamic nutrition dashboard with real-time updates

## Technologies and Libraries
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js with Express
- **Database**: PostgreSQL
- **Version Control**: GitHub
