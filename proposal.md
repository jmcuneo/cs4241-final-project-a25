## Project Description  
Our group plans to create a sports betting web application that simulates wagering on live games using virtual coins and without using real money. Users will register for accounts and receive a starting balance of virtual coins. They can place bets on real sporting events, with live odds updated through an existing FanDuel API. Winnings and losses will be reflected in each user’s coin balance.  

The app will feature authentication with username and password, as well as a personalized dashboard showing betting history, current wagers, and available balance. All data, including login credentials, bets placed, and coin totals, will be stored in a persistent MongoDB database. This project aims to demonstrate web development concepts such as responsive UI, API integration, dynamic state management, and backend persistence, while also being engaging for users.  

## Key Features  
- User authentication with account creation and login.  
- Virtual currency system for betting.  
- Live sports odds pulled from FanDuel’s API.  
- Dashboard showing active bets, results, and transaction history.  
- Responsive and accessible UI designed with Tailwind.  

## Technologies and Libraries  
- **Frontend**: React (with JavaScript or TypeScript), Tailwind CSS for styling.  
- **Backend**: Node.js with Express for server-side logic.  
- **Database**: MongoDB for persistent storage of users, bets, and coin balances.  
- **External API**: FanDuel (for live odds).
